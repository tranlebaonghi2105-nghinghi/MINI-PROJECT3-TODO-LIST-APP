# MINI PROJECT 3 - TO-DO LIST MANAGEMENT

## Project Description
This is a Python console application for managing a personal to-do list.
The program allows users to add, display, search, sort, and save tasks.

## Features
- Add a new task
- Display all tasks in table format
- Search task by ID
- Sort tasks by title
- Show task statistics
- Save tasks to TXT file
- Load tasks from TXT file
- Search tasks by keyword
- Export tasks to JSON

## Technologies Used
- Python
- Text File (.txt)
- JSON File (.json)
- Git & GitHub
